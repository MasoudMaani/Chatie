## Genesis

There exists a certain theological claim that Arabic is the language of God. The reasoning is simplistic: Arabic, a Semitic-rooted language, takes three-consonant roots and drops them into pre-made algebraic matrices of vowels and consonants, generating infinite words. Most are useless and mean nothing. It's bloated, inefficient, and by some illiterate reasoning, divine.

Here's the thing: **The Persians invented algebra.** If using algebra makes a language divine, then technically the Persians are the God of Gods, and we're also Gods now because we just built the same system in ten minutes on Indo-European-rooted English and without the need for pre-made matrices.

The claim "Arabic is God's language" collapses the moment you realize: **the divinity was never in Arabic. It was in the algebra all along.**

And algebra belongs to everyone.

## What Is Chatie?

Chatie is a Python script that generates synthetic English words following natural phonotactic patterns. It respects the morphology of English, understands consonant clusters, respects vowel sequences, and produces pronounceable output bounded between 2 and 15 characters.

But more importantly: **Chatie is honest about what it does.** It creates. It speaks in known and alien tongues without making a single apology.

### The Rules

- **Pronounceability first**: Words follow natural syllable structures (CV, CVC, CCV, CCVC, VC, V) with valid English onset and coda clusters.
- **No chaos**: Consonant clusters must be phonotactically real; no more than 2 consecutive vowels; diphthongs are recognized and respected.
- **Size matters**: Words range from 2 to 15 characters—long enough to be interesting, short enough to be useful.
- **Honesty**: They might sound vaguely familiar. They might sound alien. They're never pretending to be more than they are.

## Usage

### Generate a single word

```python
from chatie import WordGenerator

gen = WordGenerator()
word = gen.generate_word()        # Random length (2-15)
word = gen.generate_word(7)       # Exactly 7 characters
```

### Generate a batch

```python
words = gen.generate_batch(50)              # 50 words, random lengths
words = gen.generate_batch(20, (3, 8))      # 20 words, 3-8 characters each
```

### Run the test suite

```python
from chatie import TestWordGenerator

tester = TestWordGenerator()
tester.run_all_tests()
```

## Technical Details

**Phoneme Inventory:**
- Vowels: a, e, i, o, u
- Consonants: b, c, d, f, g, h, j, k, l, m, n, p, q, r, s, t, v, w, x, y, z
- Diphthongs: ai, au, ea, ei, eu, oa, oi, ou
- Valid clusters: 37 onset clusters, 34 coda clusters (based on English phonotactics)

**Constraints:**
- Length: 2-15 characters
- Consecutive consonants: Must form valid clusters or be separated by vowels
- Consecutive vowels: Max 2 (excluding diphthongs)
- Syllable structures: Randomly chosen from natural patterns

**Testing:**
- Length bound validation
- Pronounceability verification
- Cluster validity checking
- Batch generation integrity
- Edge case handling
